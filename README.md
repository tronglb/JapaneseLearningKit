# Japanese Learning Kit

Personal Japanese study kit from N5 to N2.

## Current version
- N5 Lesson 10
- Offline flashcards with Japanese speech using Mac browser speech synthesis
- CSV vocabulary database
- Anki TSV import file

## Quick start
1. Open `lessons/lesson10/flashcards.html` in Chrome or Safari.
2. Study 10–15 minutes.
3. Use arrow keys to move cards, Space to flip.
4. Use `database/vocabulary.csv` as the master vocabulary database.
